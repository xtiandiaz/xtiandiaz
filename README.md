Formerly a native iOS Developer who migrated (back) to **Web & Game Dev.**, and to cultivate a passion for digital illustration, **UI & Graphic Design**, with the aim to extend the scope, variety and fun of my career.

# Some open-source work
You're welcome to explore a bunch of my public repos, most of them recently updated, containing either prototypes, utilities or even frameworks (or at least attempts) done as exercises to learn and to develop something atypical but still meaningful to share:

## Utilities
- [Design Tokens](https://github.com/xtiandiaz/design-tokens): scripts using Node and the Figma API to read the design tokens (icons, colors, text-styles) from a public Figma 'file', and generate code to use them (on the Web).

## Frameworks
- [Emerald](https://github.com/xtiandiaz/emerald): a simple game framework, built on top of PixiJS for rendering, supporting custom 2D physics, meant as another alternative to develop games on WebGL.
- [Vueties](https://github.com/xtiandiaz/vueties): a collection of Vue components shared by several of my tiny Web-apps (listed below), and using the Design Tokens described above.

## Vue tiny SPAs
- [Ari](https://github.com/xtiandiaz/ari): a simple Web-app to exercise mental math.
- [Recetario](https://github.com/xtiandiaz/recetario): another to store and share cooking recipes in a simplified way.
- [Lexi](https://github.com/xtiandiaz/lexi): another to play guessing random words (in several languages) by ordering their shuffled letters and (then getting access to a bunch of links to learn more about them).

## Unity prototypes
- [Buc-card-neer](https://github.com/xtiandiaz/buc-card-neer): a card-game prototype built with Unity, structured as a gamified app using the MVVM architecture, and employing plugins such as Zenject (for dependency management) and UniRx (for reactivity).
- _Will dust off and add more soon!_

## Swift-UI game prototypes
- [Card Hunter](https://github.com/xtiandiaz/card-hunter-swiftui): a game-like prototype to move around a schematic dungeon using view-cards.
- [Cardibbean](https://github.com/xtiandiaz/cardibbean-swiftui): another simple but potentially useful prototype to drag and drop (and pile) cards around on a view-canvas.
- [Sokomatch](https://github.com/xtiandiaz/sokomatch-swiftui): a more ambitious game-like prototype, to swipe and move a player's token around a grid stage, meaning to solve generative puzzles using the [Sokoban](https://en.wikipedia.org/wiki/Sokoban) mechanics.

## Misc. archived repos
- [Kids Puzzle Adventure](https://github.com/xtiandiaz/kpa): a simple but cute jigsaw puzzle game app, built long-ago with PixiJS, and formerly sold on Envato's Codecanyon.
